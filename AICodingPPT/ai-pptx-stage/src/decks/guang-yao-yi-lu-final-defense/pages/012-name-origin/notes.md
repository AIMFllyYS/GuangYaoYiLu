# 012 “光药医路”：三个支部各取一字，走成一条路

## 本页叙事目标

“路”代表从校园出发的实践路径，也代表中医药文化在青年群体中的传播路径。

## Morph 动效意图

- 复用 `!!brand-logo-primary`、`!!brand-logo-secondary`、`!!deck-title`、`!!herbal-ring`、`!!gold-path`，让品牌、标题与光路在章节之间平滑移动。
- 本页照片流使用 `!!photo-strip-01` 至 `!!photo-strip-06`，可在 PowerPoint 中做照片滑入、缩放、换位与幕布收束。
- 本页模块卡使用与章节对应的稳定 `!!` morphKey，强调从上一页信息结构自然变形到当前重点。

## PPTX 手工复刻注意事项

- 背景图是无文字 bitmap，所有中文标题、说明、页码、卡片文案都必须保留为真实文本框。
- SVG 图标通过 image 图层插入，迁移到 PowerPoint 后可按需替换为内置图标，但不要合并成整页截图。
- 同一页内不要复制出重复 `morphKey`；跨页复刻 Morph 时保持同名对象一一对应，且对象类型不变。
