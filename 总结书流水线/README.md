# 总结书流水线

[**English**](README.en.md)

本目录汇集 **多渠道、完整书名称**（约 CH01 封面～CH15 封底）的草稿与排版资产，便于与 [`输出终稿/`](../输出终稿/)（若存在）择优合并。具体 **页面尺寸、风格与编造内容边界**见根目录 [**Plan.md**](../Plan.md)。

## 子目录

| 路径 | 说明 |
|------|------|
| [`Antigravity/`](Antigravity/) | 主渠道之一：逐章文件夹 `CH01_封面` … `CH15_封底`，各章常见 `brief.md`、`config.json`、插图与版面资源；`_preview/` 为分页预览，`_shared/` 为跨章素材；另有 `compositor.py`、`export_pdf.py`、`renumber.py` 等辅助脚本 |
| [`NotebookLM/`](NotebookLM/) | NotebookLM 侧提示词或导出（例如 [`PPT提示词合集.md`](NotebookLM/PPT提示词合集.md)） |

若你本地曾有过 **Google Flow** 等其他渠道产物，可归入同源「全书」类比；**不必强求目录名一致**，以本章说明与 [`Plan.md`](../Plan.md) 为准。

## 与素材、终稿的关系

```text
素材库 ──► 总结书流水线（多渠道各自「全书」）──► 输出终稿（择优合并）
```
