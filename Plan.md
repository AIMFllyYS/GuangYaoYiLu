## 三、设计规范

### 参考作品风格解析

优秀参考作品（132页）采用 **手绘漫画 + 真实照片嵌入** 的混合风格：

| 元素 | 参考作品做法 | 我们的策略 |
|------|-----------|---------|
| 章节封面 | 深蓝科技风数字海报 | AI生成精美数字海报 |
| 内容页 | 手绘漫画 + 手写文字 + 照片嵌入 | AI插画 + 手写体 + 真实照片 |
| 文字排版 | 大量手写体，标题艺术字 | 正文印刷体，感悟手写体 |
| 照片处理 | 直接贴入手绘页面 | 照片融入AI海报版式 |
| 页码 | 右下角 `X/132` | 右下角 `X/N` |

### 差异化优势

往年**手绘**为主（画得多=用心），今年是**电子版**：
1. AI生成高品质插画，质感远超手绘
2. 动态海报/动画版本，在文件形式上创新（学姐建议的加分项）
3. 看板娘角色贯穿全书，增加辨识度
4. 信息图/地图等数据可视化，展示专业度

### 视觉风格
- **不是手绘风**
- AI 高品质数字插画海报 + 真实照片融合排版
- 色彩体系：三色主线贯穿——白（破冰）、红（思政）、绿（中医药）
- 中医药元素：草药纹理、药壶、古籍、中药材装饰贯穿全书

### 字体规范

| 用途 | 字体选择 | 说明 |
|------|---------|------|
| 正文介绍 | 印刷体（思源黑体/等线/宋体） | 清晰易读 |
| 章节大标题 | 艺术字/书法体 | 视觉冲击力 |
| ⭐ **同学感悟** | **手写体** | 🔑 模拟同学亲笔便签条 |
| ⭐ **团支书感悟** | **手写体** | 建议每人用不同手写体 |
| 英文装饰 | 英文衬线体 | 点缀用 |
| 数据/图表 | 等宽/无衬线 | 信息图 |

### 图文比例
- 每页**图片面积 ≥ 60%**，文字精炼
- 活动照片大图展示，避免缩成邮票
- 每页通常 1-3 张照片 + 少量文字

## 五、逐章产出详细规格

### CH01 封面（1页）
- **内容**：支部名称 + 活动主题（十四字）+ 看板娘形象 + 年份
- **设计**：三色渐变背景，中医药元素装饰
- **要素**：看板娘需与CH04统一，三专业象征元素（光学透镜/药草/DNA）

### CH02 目录（1-2页）
- **内容**：全书章节导航，三色色块区分叙事线
- **设计**：时间轴/路径式排版，**不能是纯文字列表**

### CH03 主题介绍（3-4页）
- **内容**：十四字主题 + 选题初心 + "光药医路"名称释义
- **素材**：`素材库/02总策划书/`
- **文案**：AI从策划书提炼润色

### CH04 看板娘（2-4页）
- **⭐ 全书灵魂角色，需最先确定**
- 页面1：全身高清大图
- 页面2：人物设定介绍（姓名、性格、象征意义）
- **角色**：融合三专业——手持草药（药学）+ 透镜装饰（光电）+ DNA/白大褂（基医）
- **画风**：日系立绘 × 中国风

### CH05 三个支部介绍（4-6页）
- 每支部1-2页：合照 + AI编写介绍词
- **素材**：`素材库/08人员与合照/`（三张合照）

### CH06 人员分工（3-4页）
- 组织架构图 + 人员名单 + 工作介绍词
- **素材**：`08总结书-联合支部人员统计.xlsx`

### CH07 初识白·破冰（8-12页）🤍
- 破冰活动介绍 + 12张照片 + **编造感悟**
- **素材**：`素材库/03破冰活动/`
- ⭐ 感悟：3-5条，人名取自人员统计表，**手写体**便签条样式

### CH08 思政红·团课（6-8页）❤️
- 两次团课：十五五规划 + 3.29主题团会
- **素材**：`素材库/06三月活动/`（PPT + 照片）
- ⭐ 感悟：2-3条，手写体
- 强调"由团组织联系起来"的纽带意义

### CH09 思政红·志愿服务（10-15页）❤️
