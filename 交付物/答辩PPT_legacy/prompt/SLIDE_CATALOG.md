## 设计类型说明

| 代码 | 类型 | 特征 |
|---|---|---|
| COVER | 全艺术封面 | 大书法标题+人物插画+山水背景，无照片框 |
| INFOGRAPH | 信息图 | 标题+圆形/流程节点图示 |
| CHARACTER | 角色展示 | 人物立绘+符号图示网格 |
| PRODUCT | 产品陈列 | 产品横向排列，下方标签 |
| VIDEO | 视频框页 | 大型视频占位框，装饰边缘 |
| FLOW | 叙事弧/时间轴 | 彩色流线弧，节点标记 |
| PHOTO | 照片排版 | 明确的照片留白框（1大+N小 或 等宽） |
| MIXED | 混合版式 | 照片框+信息图+实景背景混排 |
| MAP | 地图信息图 | 中国地图+省份标注+照片框 |
